# sysops-scripts

個人用的系統管理 script 集合，依功能分資料夾存放。

---

## 結構

```
sysops-scripts/
└── nas/
    ├── disk_info.sh    # 查詢硬碟容量、序號、所屬 MD RAID
    └── ...
```

各資料夾內有獨立的 README 說明。

| 資料夾 | 說明 | 連結 |
|--------|------|------|
| `nas/` | NAS 相關 script | [前往 GitHub](https://github.com/eric90040406/sysops-scripts/tree/main/nas) |

---

## 環境需求

- OS: Linux (Debian / Ubuntu)
- Shell: Bash 4.0+
- 大部分 script 需要 `root` 權限
